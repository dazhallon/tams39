We can check if the matrices can be pooled by using the test variable
\begin{equation*}
  \lambda^* = \frac{\Pi \det(V_i ^{f_i/2})}{\det(V^{f/2})}
  \frac{f^{pf/2}}{\Pi f_i ^{pfi/2}}
\end{equation*}
and using the Box correction
\begin{equation*}
  -2 * \frac{m}{f} \ln \lambda^*.
\end{equation*}
